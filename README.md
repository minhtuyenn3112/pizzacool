# Cách tải source code về máy tính của bạn

Vào link github mà tôi gửi bạn sau đó ấn vào ô CODE sao chép link có dạng https://abc123456.git

Vào VS code bấm vào Terminal -> New terminal -> Sau đó gõ lệnh cd "Thư mục mà bạn muốn lưu source code"

Tiếp theo gõ lệnh git clone "cái link mà bạn đã sao chép trong github" là tất cả source code lưu về thư mục đó

Vào terminal gõ cd backend nhập lệnh "npm install" sau đó gõ lệnh "npm run dev" để chạy backend

Vào terminal gõ cd frontend nhập lệnh "Remove-Item package-lock.json -Force", "npm install", chạy "npm run dev" để chạy web
